# 택시 미터기 - 법적 문서

이 폴더에는 앱스토어 제출에 필요한 법적 문서들이 포함됩니다.

## 📄 포함된 문서

- **index.html** — 한국어/영문 개인정보 처리방침 (Privacy Policy)
  - 한국어 및 영문 지원
  - 모바일/데스크톱 반응형 디자인
  - GitHub Pages를 통해 자동 호스팅

## 🌐 접근 방법

GitHub Pages가 활성화되면 다음 URL에서 접근 가능합니다:

```
https://<username>.github.io/taxi-meter-app/
```

예시: `https://jeseon.github.io/taxi-meter-app/`

## ⚙️ 설정 방법

1. GitHub 리포지토리 이동
2. **Settings** → **Pages**
3. **Source** 선택:
   - Branch: `main` (또는 `master`)
   - Folder: `/ (root)` 또는 `/docs`
4. **Save** 클릭
5. 1-2분 후 GitHub Pages URL 확인

## 📝 앱스토어 제출 시 사용

### iOS App Store
App Store Connect → 앱 정보 → 개인정보 보호정책
```
https://<username>.github.io/taxi-meter-app/
```

### Android Google Play
Google Play Console → 앱 설정 → 개인정보 보호정책
```
https://<username>.github.io/taxi-meter-app/
```
