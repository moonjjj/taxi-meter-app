🚕 택시 미터기

개인정보 처리방침
마지막 업데이트: 2026년 5월 1일

1. 개요

「택시 미터기」(이하 "앱")은 사용자의 GPS 데이터를 활용하여 이동 거리와 소요 시간을 측정하고 요금을 계산하는 서비스를 제공합니다. 본 개인정보 처리방침은 앱이 어떤 정보를 수집하고, 어떻게 사용하며, 어떻게 보호하는지 설명합니다.

2. 수집하는 개인정보

2.1 GPS/위치 데이터

  • 위도, 경도, 고도 — 사용자의 현재 위치
  • 이동 속도 — 실시간 속도 측정
  • 수집 시점 — 앱 실행 중일 때만

2.2 수집되지 않는 정보

  • 개인 신원 정보 (이름, 전화번호, 이메일 등)
  • 기기 식별자 또는 추적 ID
  • 사용 기록 또는 분석 데이터
  • 제3자와의 통신 기록

3. 개인정보의 사용 목적

  • 요금 계산 — 이동 거리 및 소요 시간 기반 택시 요금 계산
  • 속도 측정 — 거리 모드(≥4 km/h)와 시간 모드(<4 km/h) 자동 전환
  • 사용자 경험 개선 — 앱 기능이 정상 작동하도록 보장

4. 데이터 보관 및 처리

4.1 저장 위치

  • 모든 위치 데이터는 사용자 기기의 메모리에만 저장
  • 외부 서버 또는 클라우드로 전송되지 않음
  • 앱 종료 시 자동으로 삭제됨

4.2 데이터 처리

  • GPS 신호는 정확도 향상을 위해 평활화(스무딩) 처리
  • 이상 데이터는 감지되어 제거
  • 사용자의 기기에서만 계산 수행

4.3 보관 기간

  • 앱 실행 중: 실시간 위치 데이터 사용
  • 앱 종료 시: 모든 데이터 삭제
  • 장기 보관 없음

5. 제3자와의 정보 공유

택시 미터기는 수집한 개인정보를 제3자와 공유하지 않습니다. 다음의 경우를 제외합니다:

  • 사용자의 명시적 동의
  • 법령에 따른 의무 (수사 협력 등)
  • 앱 스토어 제공자의 필수 정책 (Apple, Google)

6. 사용자의 권리

6.1 위치 권한 거부

  • iOS: 설정 → 개인정보 보호 → 위치 서비스 → 택시 미터기 → "사용 중일 때"로 설정 또는 "허용 안 함"
  • Android: 설정 → 앱 및 알림 → 앱 권한 → 위치 → 허용 안 함

6.2 권한 거부 시 영향

  • 앱이 GPS 데이터를 수집할 수 없어 요금 계산 불가
  • 시간 모드로 자동 전환 (수동 시간 입력 필요)

6.3 정보 열람 및 삭제

  • 실시간 수집 데이터는 기기의 설정에서 언제든 확인 가능
  • 앱 제거 시 모든 저장된 데이터 완전 삭제

7. 보안

  • 모든 데이터는 기기 내에서만 처리되므로, 전송 중 데이터 유출 위험 없음
  • 암호화 또는 특수 보안 기술 필요 없음 (기기 내 처리)
  • 기기의 운영 체제 보안 메커니즘으로 보호

8. 아동 개인정보

본 앱은 특정 연령 이상을 대상으로 합니다. 보호자는 아동의 위치 권한을 기기 설정에서 제어할 수 있습니다.

9. 정책 변경

이 개인정보 처리방침은 언제든 변경될 수 있습니다. 중대한 변경 사항은 앱 업데이트 또는 이 페이지 공지를 통해 알립니다. 지속적인 사용은 업데이트된 정책에 동의하는 것으로 간주됩니다.

Privacy Policy
Last Updated: May 1, 2026

1. Overview

「Taxi Meter」(hereafter "App") provides a service to measure travel distance and time using GPS data, and calculates fare based on the data. This Privacy Policy explains what information the App collects, how it is used, and how it is protected.

2. Information We Collect

2.1 GPS / Location Data

  • Latitude, Longitude, Altitude — Your current location
  • Speed — Real-time movement speed
  • Collection Time — Only when the App is running

2.2 Information We Do NOT Collect

  • Personal identity information (name, phone, email, etc.)
  • Device identifiers or tracking IDs
  • Usage history or analytics data
  • Communication records with third parties

3. Purpose of Information Use

  • Fare Calculation — Calculate taxi fare based on distance traveled and time spent
  • Speed Detection — Automatically switch between distance mode (≥4 km/h) and time mode (<4 km/h)
  • Service Improvement — Ensure the App functions properly

4. Data Storage and Processing

4.1 Storage Location

  • All location data is stored only in the user's device memory
  • Not transmitted to external servers or cloud services
  • Automatically deleted when the App closes

4.2 Data Processing

  • GPS signals are smoothed to improve accuracy
  • Anomalous data is detected and removed
  • All calculations performed only on the user's device

4.3 Retention Period

  • During App Usage: Real-time location data is used
  • When App Closes: All data is deleted
  • No long-term storage

5. Sharing with Third Parties

Taxi Meter does not share collected personal information with third parties, except in the following cases:

  • With your explicit consent
  • As required by law (law enforcement cooperation)
  • Required by App Store providers' policies (Apple, Google)

6. Your Rights

6.1 Location Permission Control

  • iOS: Settings → Privacy → Location Services → Taxi Meter → "While Using" or "Never"
  • Android: Settings → Apps & Notifications → App Permissions → Location → Don't Allow

6.2 Impact of Denying Permission

  • The App cannot collect GPS data and cannot calculate fare automatically
  • Automatically switches to time-only mode (manual time input required)

6.3 Access and Deletion

  • Real-time collected data can be reviewed in your device settings at any time
  • All stored data is permanently deleted when the App is uninstalled

7. Security

  • All data is processed only on your device, so there is no risk of data leakage during transmission
  • No encryption or special security technology needed (on-device processing)
  • Protected by your device's operating system security mechanisms

8. Children's Information

This App is intended for users of a certain age and above. Parents/guardians can control location permissions for children through device settings.

9. Policy Changes

This Privacy Policy may be updated at any time. Material changes will be notified through App updates or notices on this page. Continued use of the App constitutes acceptance of the updated policy.
